# Referência técnica do projeto

## Objetivo atual

- Migrar o site original em Joomla/YOOtheme para o tema WordPress deste repositório.
- `https://locutora.com` é a referência visual e de conteúdo do site original.
- `https://plum-walrus-211817.hostingersite.com` é o ambiente WordPress temporário que deve reproduzir a referência.
- O domínio temporário anterior `darkgoldenrod-swan-951169.hostingersite.com` foi desativado e não deve receber novos deploys.
- Em comparações, sempre corrigir o ambiente WordPress; não tratar a URL temporária da Hostinger como referência do design.

## Escopo de fidelidade

- Comparar desktop e mobile, incluindo cabeçalho, espaçamentos, tipografia, imagens, vídeos, cores, recortes, animações, rodapé e botão flutuante.
- Preservar o conteúdo editável do WordPress quando isso não alterar o resultado visual.
- Usar os arquivos públicos do site original apenas como referência; implementar e manter os arquivos no tema local.

## Verificação

- Validar PHP e JavaScript modificados.
- Verificar visualmente as páginas no ambiente WordPress após alterações de layout.
- Não considerar a migração concluída enquanto diferenças visuais relevantes permanecerem.
